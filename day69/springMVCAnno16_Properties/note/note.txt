  
  -- Properties -- 
  
    (웹)애플리케이션을 개발할 때 프로그램 실행 중
    변하지 않고 고정되어있어야 하는 값들을 작성함
    
    Spring MVC 에서는 이러한 값들을
    properties 파일에 작성하고
    사용할 수 있도록 제공함
    
    @PropertySource, @PropertySources 를 사용함
    하나 또는 여러 개의 properties 파일을 지정할 수 있음
    
    @Value
      properties 파일에 작성한 값을 주입 받을 수 있음
